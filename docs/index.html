What is this?

We ran the 38 theses from the official Bundestagswahl 2025 Wahl-O-Mat through five frontier LLMs and recorded their answers (AGREE / NEUTRAL / DISAGREE). We then fed each model's answers into the Wahl-O-Mat browser interface via Playwright automation to get official party alignment scores.

Models evaluated

  • GPT-4o — OpenAI (provider default temperature)
  • o3 — OpenAI reasoning model
  • Claude Opus 4.7 — Anthropic
  • Claude Sonnet 4.6 — Anthropic
  • Claude Haiku 4.5 — Anthropic

Methodology

Each model received a system prompt instructing it to respond with exactly one word: AGREE, NEUTRAL, or DISAGREE. Theses were presented independently (no conversation history carried across theses). Responses were mapped: AGREE → +1, NEUTRAL → 0, DISAGREE → −1.

Party alignment scores (0–100%) are computed by the official Wahl-O-Mat algorithm via Playwright browser automation — not a custom metric.

Key findings

  • Tierschutzpartei ranks #1 across 4 of 5 models (avg 80.3%), driven by its animal welfare, climate, and social equity platform.
  • AfD ranks last in every model (avg 29.0%), with no model scoring it above 38%.
  • Left-progressive tilt is robust across both model families — SPD, Volt, GRÜNE all cluster in the top 8 for every model.
  • CDU/CSU ranks 22nd (avg 43.7%), 33.7 pp below SPD.
  • o3's high neutral rate (37%) compresses its party scores toward 50% vs GPT-4o's more opinionated 55% agree rate.

Limitations

  • Single-pass prompting — no cross-thesis context, no multi-seed variance estimation.
  • Temperature not pinned — results may vary between runs.
  • Wahl-O-Mat scoring algorithm is proprietary — we don't control its weighting.
  • Gemini models excluded — safety filters blocked all political theses.
